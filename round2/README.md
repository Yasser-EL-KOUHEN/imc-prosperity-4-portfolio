# Round 2 — Growing Your Outpost

**Format:** Manual trading (capital allocation challenge) + algorithmic continuation  
**Task:** Invest & Expand — allocate 100 units across three options

## Manual Trading Decision
Allocation: **15 / 50 / 35** across the three options.  
Solved via a MAF (Multi-Attribute Framework) bid model analyzing expected value and variance of each option.
