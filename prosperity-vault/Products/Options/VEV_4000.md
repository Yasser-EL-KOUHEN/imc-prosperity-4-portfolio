## Deep ITM Behaviour

VEV_4000 trades with a wide ~21 spread and active flow (128–172 trades/day). With delta ≈ 1, it behaves almost like the underlying for hedging purposes — the VEV underlying hedger absorbs any directional exposure created by VEV_4000 takes.

## Quoting Strategy (`vault/round4_trader.py`)

Two-sided market making, allows naked short:

```python
"VEV_4000": {
    "passive_size": 6, "take_buy_edge": 14.0, "take_sell_edge": 7.0,
    "passive_buy_edge": 9.0, "passive_sell_edge": 5.0,
    "allow_naked_short": True, "two_sided": True,
    "obi_beta": 0.0, "bid_only": False, "sell_only": False,
}
```

The wide edges (`take_buy_edge=14`, `passive_buy_edge=9`) reflect the deep-ITM character — fair value moves are dominated by the underlying, not by IV changes.
