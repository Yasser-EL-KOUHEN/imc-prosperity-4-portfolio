## Quoting Strategy: Two-Sided Passive

VEV_5300 uses **two-sided quoting** — we post both a bid and an ask:

```
fair_call = BS(spot_fair, 5300, T, sigma_live)
fair_call += obi_beta * OBI(order_depth)  # obi_beta = 0.65

bid = round(fair_call - passive_sell_edge)
ask = round(fair_call + passive_sell_edge)
```

We quote passively (inside the spread) and wait for fills.
